#sfxvi_documents

1. system_work_area.txt

キャラクタやサブシステムから参照可能なシンボル_sfsysInfoに格納される値についてまとめたものになります。

2. player_work_area.txt

キャラクタの情報が格納されるワークエリアについてまとめたものになります。

3. sfxvi.s

SFXVI本体をディスアセンブルしたソースファイルですが、ロジックを解析しC言語に置き換えを進めている最中のものです。
ただし何をやっているかを明らかにすることを主眼に置いているため、正しいC言語の文法には則っていません。

各種シンボルの命名について

・SYSCALL_xxxx
システムコールサブルーチンです。APIから呼ばれます。

・SFSYSINFO_xxxx
システムワークエリアです。上記のsystem_work_area.txtに一致します。

・WORK_1P_xxxx, WORK_2P_xxxx, WORK_3P_xxxx, WORK_4P_xxxx
キャラクタワークエリアです。上記のplayer_work_area.txtに一致します。
内部的には4キャラ分のワークが存在しますが4人同時プレイのためのものではなさそうです。

・SR_xxxx
システムコールではないサブルーチンです。

・XCLIB_xxxx
XCコンパイラ付属のライブラリからリンクされたと推測されるサブルーチンです。
今後、ソースからは削除する予定です。
